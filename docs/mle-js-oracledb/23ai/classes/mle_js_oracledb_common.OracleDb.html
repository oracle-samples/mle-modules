defaultConnection

  • defaultConnection(): IConnection

  • Returns the default connection object for executing SQL queries in the Oracle Database using mle-js-oracledb. Note that with MLE, because JavaScript is executed directly in the database, there is no need to establish a specific connection, which is why the default connection object should be used.

    Returns

    default connection object for executing SQL queries with mle-js-oracledb.
