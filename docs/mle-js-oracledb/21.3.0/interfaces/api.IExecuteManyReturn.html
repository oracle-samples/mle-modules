Optional outBinds

outBinds?: (any[] | Record<string, any>)[]
  • Defined in api.ts:345

This contains the value of any returned IN OUT or OUT binds. It is an array of arrays, or an array of objects, depending on the {@link binds} parameters structure. The length of the array will correspond to the length of the array passed as the binds parameter. It will be present only if there is at least one OUT bind variable identified.

Optional rowsAffected

rowsAffected?: number
  • Defined in api.ts:352

This is an integer identifying the total number of database rows affected by the processing of all records of the binds parameter. It is only present if a DML statement was executed.
