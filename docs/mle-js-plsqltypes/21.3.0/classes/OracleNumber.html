shift

  • shift(digits: number | OracleNumber): OracleNumber
    • Defined in mle-js-plsqltypes.ts:395

    Shift the number by the specified number of decimal places

    Parameters

    • digits: number | OracleNumber

      number of decimal places to shift. Can be negative. Positive values shift the decimal place to the right and negative values to the left. For example, if NUMBER corresponds to 1234.5 and digits == -1, the new NUMBER object will correspond to 123.45.

    Returns OracleNumber

    an Oracle number containing the shifted result
