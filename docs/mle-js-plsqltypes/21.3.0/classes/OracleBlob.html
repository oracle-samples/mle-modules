getChunkSize

  • getChunkSize(...args: any[]): number
    • Defined in mle-js-plsqltypes.ts:845

    When creating the table, you can specify the chunking factor, a multiple of tablespace blocks in bytes. This corresponds to the chunk size used by the LOB data layer when accessing or modifying the LOB value. Part of the chunk is used to store system-related information, and the rest stores the LOB value. This method returns the amount of space used in the LOB chunk to store the LOB value.

    • Defined in mle-js-plsqltypes.ts:913

    This method reads a piece of a LOB, and returns the specified amount into the buffer parameter, starting from an absolute offset from the beginning of the LOB. If the input offset points past the End of LOB, a NO_DATA_FOUND exception is raised.

    Parameters

      Number of bytes to read.

    • offset: number

      Offset in bytes from the start of the LOB (origin: 1).

    Returns Uint8Array
